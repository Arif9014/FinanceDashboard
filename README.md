# FinanceDashboard
Project Overview:

This project is an interactive Personal Finance Dashboard built using Power BI, aimed at visualizing and analyzing financial data across multiple years. The dashboard helps users track income, expenses, savings, and growth trends, enabling data-driven financial insights through a clean and user-friendly interface.

Key Features:

Dynamic KPI Cards displaying total Income, Expenses, Savings, and Savings Percentage.
Time-based analysis with year-wise and month-wise slicers (2021â€“2024).
Trend analysis using line charts to track expense behavior over time.
Category-wise breakdown of Expenses and Savings using donut charts.
Monthly growth indicator to monitor financial performance trends.
Interactive navigation using Power BI bookmarks and buttons for enhanced UX.

Tools & Techniques:

Power BI Desktop.
DAX measures for KPIs and percentage calculations.
Bookmarks & Buttons for navigation.
Data modeling & relationships.
Custom visual formatting for a modern dashboard UI.

Insights Delivered:

Identified major expense contributors such as rent, groceries, and EMIs.
Analyzed savings distribution across different investment components.
Monitored financial growth trends over time to evaluate performance.
Enabled quick comparison of income vs expenses for better budgeting decisions.

Outcome:

The dashboard demonstrates the ability to transform raw financial data into clear, actionable insights through effective data visualization and interactivity, showcasing practical Power BI skills relevant to real-world business scenarios.
